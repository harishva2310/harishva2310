System Design of the Vehicle Rental App

Technologies Used:

  1. Front-End Service: ReactJS, Typescript, NodeJS, JavaScript, Axios, Bootstrap CSS, Mobirise Custom CSS
  2. Back-End Service: Spring, Spring Boot, Spring Security, JWT, Spring Data REST, Hibernate
  3. Cloud Technologies: Oracle Cloud Infrastructure, Oracle Autonomous Database, Oracle Kubernetes Engine, Oracle Compute Instances, Docker, Nginx
  4. Authentication: Okta Identity Engine, Okta OAuth2.0, SPA Authentication
  5. Payments: Stripe API
